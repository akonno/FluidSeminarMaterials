工学院大学 工学部機械工学科 機械工学セミナー（流体工学セミナー）

このサイトは「流体工学セミナー」で使用している Python 演習教材をまとめたものです。

  • Miniforge ではじめる Python プログラミング

  • JupyterLab の導入と基本操作

  • PyGame-CE を用いたインタラクティブ表現・アニメーション入門

プログラミングの基礎から、図形描画や簡単なゲーム制作までを扱います。
現在、内容を順次整備中です。このページはトップページです。（未完成）

教材内の表示について¶

ポイント

「ポイント」は特にプログラムの説明に使われています．

演習課題

「演習課題」は実際にプログラムを書いて試してみてほしい課題を記載しています．ただし提出を求めている課題ではありません．

課題

「課題」（演習課題ではない）は提出を求めている課題です．
